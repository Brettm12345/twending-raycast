# Twending

A new way to view github trending

## Configuration Options

- `defaultLanguage`: The default language to use if skip language selection is enabled
- `defaultPeriod`: The default period to use if skip period selection is enabled
- `githubToken`: The personal access token to use to access the GitHub API
  - This is optional, but it is recommended to set it to your personal access token.

## Commands

- `view-default-language`: View trending repositories for the default language
- `view-trending-repositories`: View trending repositories for a specific language
